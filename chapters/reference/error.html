Error ClassesÔÉÅ

This implementation provides individual error classes for all errors defined by the Erbsland Configuration Language. They all share a common base class Error, which allows a convenient ‚Äúcatch-all‚Äù error handling while parsing and evaluating the configuration.

The ConfValueNotFound error also subclasses KeyError, to make value lookup match the usual Python conventions.
